# 📚 Book Management System

A clean, responsive React application built using Vite, Tailwind CSS, and standard Fetch API architecture. The application connects to a hosted cloud mock database to execute full CRUD data management cleanly.

## ✨ Core Features
- **Create:** Add new entries securely via standard modal dialog sheets.
- **Read:** Display all books with Title, Author, Genre, and Publication Year dynamically from a server.
- **Update:** Pre-populate forms to alter database records dynamically using specific resource IDs.
- **Delete:** Remove entries securely from the cloud with immediate view updates.
- **Search & Filter:** Search listings in real-time across Title/Author while dynamically matching matching Genre queries.
- **Error States:** Gracefully handled error handling with detailed network response validation fallback loops.
